A complete web server application built with Node.js, Express, and SQLite database. This project demonstrates fundamental web development concepts including server setup, routing, form handling, and database operations.

## 📋 Project Overview

This is a **Web Engineering Lab Assignment (LT1-1)** that implements:
- ✅ Web server installation and configuration
- ✅ Dynamic web page hosting
- ✅ Multi-field data entry form
- ✅ SQLite database integration
- ✅ CRUD operations (Create, Read)

## 🎯 Features

### 1. **Home Page** (`/`)
- Displays student information (Name & Registration Number)
- Clean, responsive design
- Navigation to other pages

### 2. **Data Entry Form** (`/form`)
- 4-field form (NAME, REG, DOB, EMAIL)
- Client-side validation
- POST request handling
- Success confirmation

### 3. **Database View** (`/users`)
- Displays all registered users
- Dynamic table generation
- Formatted data presentation

### 4. **SQLite Database**
- Automatic table creation
- Pre-populated dummy data
- Persistent storage
